Pénzügyi nyilvántartó rendszer webes környezetben

Az alkalmazás 2018 második félévében készült Eötvös Loránd Tudományegyetem Iinformatikai Karának Programtervező informatikus BSc végzettségének megszerzéséhez szükséges szakdolgozat keretein belül. Témája többfelhasználós pénzügyi nyilvántartási rendszer webes környezetben. Egy felhasználónak lehetősége van regisztrációt és bejelentkezést követően feljegyezni, módosítani, törölni bevételeit és kiadásait kategorizálva. Egyes kiadásait megjelölheti skálázott fontossági szempont alapján. Az elérhető pénzmozgási minták alapján különböző statisztikák grafikus megjelenítésére lehetőséget biztosít az alkalmazás havi/éves lebontásban.

A alkalmazás Java 8 nyelven íródott, építő eszköze Gradle build tool, az program magját Spring Boot keretrendszer képezi, amely integrálja a beágyazott Tomcat webszervert, előre konfigurált autentikációs és autorizációs mechanizmusát a Security modulnak. A felhasználóbarát, letisztult felületet Thymeleaf template engine segítségével biztosítja. Az adatok tárolása MySQL adatbázisban történik, Java objektumok és az adatbázis entitásai közötti leképezés Hibernate programkönyvtár segítségével történik.
